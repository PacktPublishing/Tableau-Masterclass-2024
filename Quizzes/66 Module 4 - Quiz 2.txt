Q1. Fixed LOD (Level of Detail) calculations allow us to write calculations regardless of the granularity of the view.
   TRUE/FALSE
Correct Answer: True


Q2. What is the correct syntax for a Fixed LOD calculation
A. fixed { category, subcategory : (sales) }
B. { fixed category, subcategory : sum(sales) }
C. { fixed category, subcategory : } (sales)
Correct Answer: B

A. When we want to convert a string function to a date field.
